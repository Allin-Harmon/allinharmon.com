The Fifth Annual World Children's Art Festival Website

I created this website as the final project for my Advanced CSS class IMD215.

I tried to design the site with the perspective of a child about to create something at their desk. I also made the social media icons look like stickers that you could pull off of the paper. The overall intention was to give the site a hand crafted look.

Nacho Mama's Site

I created this website as because I was interested in creating a template for menu creation.

The structure is set up so that menu items can be created and added through a content management system. The fixed nav also stay visible at 15% opacity so that any menu item can be accessed from anywhere on the page quickly.
